Dependencies
script for Windows
You need install next moduls:
- scrapy
- sqlalchemy
- bs4

Executing program
1. Open cmd 
2. go to folder with script using cd <folder path> in cmd
3. call command  scrapy crawl authors  in cd
4. now you have information in qoutes_and_authors.db file
5. You can open db usig DBevear or similar app and see information there.

Script has not add same quotes, authors and keywords. 
So when web site will be update, scripts will scrap just new information
